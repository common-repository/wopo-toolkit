Toolkit for Woocommerce, Dokan, NBDesigner,...

== Description ==

WoPo Toolkit plugin has a lot of functions help you customize your WordPress site and other plugins function. WoPo Toolkit plugin provides solutions and features for other themes or plugins:

### Wordpress

1. Disable password change  **( New )**
1. Add line break to Post's excerpt 

### Woocommerce

1. Keep only first product tab
2. Show Biographical Info and Profile Picture on My Account
3. Hide Choose an option
4. Search product only
5. Change to product search form
6. Product gallery zoom
7. Change Add to cart button to Update cart if product added to cart

### [Dokan](https://wedevs.com/dokan/?ref=658)

1. Change number vendor per page
2. Disable Ensure vendor coupon

### [NBDesigner](https://cmsmart.net/wordpress-plugins/woocommerce-online-product-designer-plugin?utm_source=wordpress&utm_medium=ref_link&utm_campaign=toolkit)

1. Show Popup after process
2. Add stylesheet inline block for NBdesigner swatch
3. Change NBdesigner main color
4. Show NBdesigner Bulk Variation vertical
5. Show NBdesigner Printing Options 2 to label
6. Move Start Design button to left side of Clear Selection button
7. Change login popup to My Account for Image upload required function
8. Scrollable printing option on Mobile device 
9. Blur license key input
10. Move Design Category to top of sidebar on templates page
11. Hide Products widget of sidebar on templates page
12. Alway on top
13. Hide process button

### [Teepro theme](https://cmsmart.net/themeforest/teepro--woocommerce-custom-t-shirt-designer-wordpress-theme?utm_source=wordpress&utm_medium=ref_link&utm_campaign=toolkit)

1. Hide User Menu when click outside
2. Add Policy Widget
3. Change single product for NBDesigner Visual Design editor layout

### [PrintShop theme](https://cmsmart.net/themeforest/printshop--wordpress-responsive-printing-theme?utm_source=wordpress&utm_medium=ref_link&utm_campaign=toolkit)

1. Show Our Capabilities on Mobile

### Tools

1. Save and restore actived plugins
2. Record updated plugins
3. View phpinfo() & imagick


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/wopo-toolkit` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'WoPo Toolkit' screen in WordPress
3. Use the WoPo Toolkit screen to configure the plugin
